Regra da cascata

A cascata é a como o CSS ler os códigos,(é lido de cima para baixo sempre),porém funciona de acordo com especificidade de seletores.

Especificidade no CSS

Especificidade é a hieraquia de pontos entre os seletores de mais especificidade para os de menos.

  • seletor pela tag: tag é normal sendo nada especifica.
  • seletor pela classe: classe direciona para eelementos presentes nela.
  • seletor pelo id: id é o mais especifico pois pode ser ultilizado apenas um por elemento.

Combinação de seletores

É a junção de dois seletores em uma linha só.

exemplos:

  • .primeiro h1: pega qualquer h1 que esteja na classe .primeiro.
  • .preimeiro > h1: o sinal de > significa que só quer pegar o h1 que é filho da clase primeiro.
  • .primeiro h1 + h1: quer selecionar apoenas o h1 que antes dels tenha h1 (só os de baixo recebe a estilização).

Pseudo-classes

É uma classe que existe no CSS mas não no HTML,e que adiciona aos seletores para definir um estado especial de um elemento.

  • first-child: pega a primeira classe.
  • last-child: pega o ultima classe.
  • nth-child(): pega a classe que ditar o número nos parenteses.
  • nth-of-type(): mesma função da classe nth-child,porém nome diferente.

Display inline e display block

  • display inline: oculpa apenas o espaço que tem mpara existir.

    • proriedades - widht,height,ppadding,margin e border.

  • display block: oculpa a largura da tela inteira.

    • propriedades - padding,margin e border (funciona apenas na horizontal).

Elementos padrãoes de cada display

  • display inline a,span,em,img e strong.
  • display block div,p,h1 á h6,header,section,main,nav,ol e ul.

Possíveis shorthand em cada propriedades.

shorthand: aceita de 1 á 4 valores para configurar lados especificos de uma vez.

  • border: aceita até 3 valores.

    • ex: border: 2px solid red;

  • margin e padding: aceita até 4 valores.

    • margin: 10px 20px 30px 40 px;

sufixos:

  • block: adiciona espaço em baixo e em cima.
  • inline: adiciona espaço nos lados.
  • top: adiciona espaço apenas em cima.
  • bottom: adiciona espaço apenas em baixo. .
  • left: espaço no lado esquerdo.
  • right: adiciona espaço no lado direito.

Box-sizing : border-box

ela respeita a altura e a largura definida,reduzindo o espaço do conteúdo.
